# Reels / Short Video (Shorts, TikTok)

**Formato:** Roteiro para vídeo curto vertical  
**Plataformas:** Instagram Reels, YouTube Shorts, TikTok  
**Duração ideal:** 30-90 segundos  
**Status:** Formato Documentado

---

## 📐 ESTRUTURA OBRIGATÓRIA

### 1. Gancho (0-3 segundos)
- Frase ou ação que **para o scroll** imediatamente
- Deve gerar curiosidade, choque ou identificação
- Nunca comece com "Olá" ou apresentações genéricas

### 2. Desenvolvimento (3-50 segundos)
- Conteúdo principal dividido em blocos curtos
- Cada bloco = 1 ideia, 1 corte visual
- Ritmo acelerado com transições visuais
- Use texto na tela para reforçar pontos-chave

### 3. Clímax / Entrega de Valor (50-70 segundos)
- O ponto principal, a revelação ou o insight mais forte
- Deve ser o momento de maior impacto

### 4. CTA / Loop (últimos 5-10 segundos)
- Call-to-action claro (seguir, comentar, compartilhar)
- OU loop para rewatches (conectar final ao início)

---

## 🏆 REGRAS DE OURO

1. **Gancho nos 3 primeiros segundos** - Se não prender aqui, perdeu
2. **1 ideia por vídeo** - Não tente cobrir múltiplos temas
3. **Texto na tela** - 70% dos usuários assistem sem áudio
4. **Ritmo rápido** - Cortes a cada 2-4 segundos mantêm atenção
5. **Formato vertical** - 9:16 sempre, nunca horizontal
6. **Linguagem falada** - Escreva como se fala, não como se lê

---

## ✅ BOAS PRÁTICAS

- Use números e listas ("3 erros que...", "5 dicas para...")
- Inclua indicações de B-roll ou corte visual entre blocos
- Indique quando usar zoom, transição ou texto animado
- Termine com pergunta para gerar comentários
- Inclua hashtags relevantes na legenda (não no roteiro)
- Música/áudio trending aumenta distribuição

## ❌ ERROS COMUNS

1. **Começar com "Oi gente"** → Use gancho direto
2. **Vídeo muito longo** → Mais de 90s perde retenção
3. **Sem texto na tela** → 70% assiste no mudo
4. **Múltiplos temas** → 1 vídeo = 1 ideia
5. **Roteiro muito formal** → Escreva como fala
6. **Sem indicações visuais** → O editor precisa saber o que mostrar
