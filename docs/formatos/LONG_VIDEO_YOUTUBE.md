# Vídeo Longo YouTube

**Formato:** Roteiro para vídeo longo  
**Plataforma:** YouTube  
**Duração ideal:** 8-25 minutos  
**Status:** Formato Documentado

---

## 📐 ESTRUTURA OBRIGATÓRIA

### 1. Hook / Abertura (0-30 segundos)
- Promessa clara do que o espectador vai aprender/ganhar
- Dado surpreendente, pergunta provocativa ou história rápida
- **Não use vinheta longa** - vá direto ao ponto

### 2. Contexto / Setup (30s - 2min)
- Por que esse tema é importante agora
- Estabeleça credibilidade rapidamente
- Apresente o framework/estrutura do vídeo ("Vou cobrir 3 pontos...")

### 3. Corpo Principal (2min - 18min)
- Dividido em **3-5 seções** claramente delimitadas
- Cada seção:
  - Título/transição visual
  - Explicação + exemplo concreto
  - Mini-conclusão ou transição
- Use timestamps para cada seção

### 4. Clímax / Insight Principal (18-22min)
- A grande revelação, conclusão ou framework completo
- Momento de maior valor - recompensa por ter assistido até aqui

### 5. CTA + Encerramento (últimos 2-3min)
- Resumo em 2-3 frases do que foi coberto
- CTA específico (inscrever, outro vídeo, comentar)
- Sugira vídeo relacionado para manter no canal

---

## 🏆 REGRAS DE OURO

1. **Hook em 30 segundos** - Promessa clara de valor
2. **Estrutura visível** - Divida em seções nomeadas com timestamps
3. **Exemplos concretos** - Cada ponto abstrato precisa de exemplo real
4. **Retenção > Duração** - Melhor 10min engajados que 25min abandonados
5. **Pattern interrupt** - Mude algo visual/tonal a cada 3-4 minutos
6. **Open loop** - Mencione algo que será revelado depois para manter audiência

---

## ✅ BOAS PRÁTICAS

- Inclua sugestões de B-roll, graficos ou tela compartilhada
- Use analogias e metáforas para conceitos complexos
- Indique momentos de ênfase visual (zoom, destaque, animação)
- Insira "checkpoints de retenção" - perguntas ao espectador
- Sugira trilha sonora/mood para cada seção
- Inclua timestamps sugeridos na descrição

## ❌ ERROS COMUNS

1. **Vinheta de 30+ segundos** → Hook direto, vinheta curta (max 5s) ou sem
2. **Monólogo sem estrutura** → Divida em seções claras
3. **Sem exemplos** → Todo ponto teórico precisa de caso real
4. **CTA genérico** → Seja específico ("comente abaixo qual dos 3 você vai aplicar")
5. **Sem indicações visuais** → Editor precisa saber o que mostrar
6. **Muito texto, pouca conversa** → Escreva como fala, não como artigo
